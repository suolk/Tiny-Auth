# 2FA-Auth-Lite - 渐进式 Web 应用

可安装到手机和桌面的 Web 应用版本。

## 在线使用

直接访问：**[https://www.netcache24.site/auth-pwa](https://www.netcache24.site/auth-pwa)**

## 安装

### Android（Chrome/Edge）

1. 访问 [https://www.netcache24.site/auth-pwa](https://www.netcache24.site/auth-pwa)
2. 点击地址栏或浏览器菜单中的**添加到手机**按钮
3. 应用将添加到你的应用程序列表

### 移动端（iOS/Android）

#### iOS（Safari）
1. 在 Safari 中打开应用链接
2. 点击**分享**按钮
3. 选择**添加到主屏幕**
4. 点击**添加**

## 使用方法

### 添加账户

1. 打开应用
2. 点击右上角的 **+** 按钮
3. 选择以下任一方式：

#### 方式 A：上传二维码图片
- 点击**上传二维码**按钮（移动端）或**扫描二维码**（桌面端）
- 选择包含二维码的截图或图片
- 密钥将自动提取

#### 方式 B：手动输入
- 粘贴**密钥**（服务提供的 Base32 字符串）
- 可选填写**用户名**、**网站名称**和**网站链接**便于管理
- 点击**保存**（或返回自动保存）

### 生成验证码

- 验证码每 30 秒自动刷新
- 点击任意验证码即可**复制到剪贴板**
- 进度条显示刷新倒计时

## 数据存储

所有数据通过 `localStorage` **本地存储**在浏览器中，不会发送到任何外部服务器。

### 数据持久性

- **桌面端**：数据持久保存，直到清除浏览器数据
- **移动端**：数据在应用安装期间持久保存

### 备份

应用不提供内置备份功能。
